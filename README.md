# Auth System

A simple authentication system project.

## Description

This repository contains the source code for an authentication system that supports user login, registration, user and session management.

## Features

- User registration
- User login
- Password hashing
- Basic session or token support

## Getting Started

1. Clone the repository.
2. Install dependencies.
3. Run the application.

## Notes

Environment Variables

Create a .env file for auth-system-backend folder:

PORT=5000
HOST=127.0.0.1
CLIENT_ORIGIN=http://localhost:3000

EMAIL_USER=your_email@example.com
EMAIL_PASS=your_email_password

DB_HOST=db_host
DB_USER=db_user
DB_PASSWORD=your_password
DB_NAME=auth_system
DB_PORT=3306

JWT_SECRET=your_secret_key
